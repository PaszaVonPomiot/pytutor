# Security Fundamentals

- Authentication: verifying identity. Who are you?
- Authorization / access control: controlling allowed actions. What can you do?
- Data or message integrity: ensuring data was not changed unintentionally. Has the data changed?
- Data or message authentication: verifying who created the data. Who authored this data?
- Nonrepudiation: preventing an actor from denying actions. Who did what?
- Confidentiality: limiting who can access data. Who can access this?
